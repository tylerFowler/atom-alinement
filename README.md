Alinement
=======================================

For those of you who like even assignment columns.

Turns this:
```javascript
const R = require('ramda');
const fs = require('fs');
const path = require('path');
const express = require('express');
```

Into this:
```javascript
const R       = require('ramda');
const fs      = require('fs');
const path    = require('path');
const express = require('express');
```
